    b) requirements.txt

Create requirements.txt:

Flask==2.2.5

c) Dockerfile

Create Dockerfile (capital D, no extension):

FROM python:3.9-slim

WORKDIR /app

# Install dependencies first (better caching)
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# Copy the rest of the app
COPY . .

EXPOSE 5000
CMD ["python", "app.py"]


3) Build the Docker image
docker build -t flask-docker-app:latest .


You should see steps complete without errors.

4) Run the container locally

If port 5000 is free:

docker run --name flask_docker_c1 -p 5000:5000 flask-docker-app:latest


Open in browser:

http://localhost:5000 (or http://localhost:5050 if you used 5050)
